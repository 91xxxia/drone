# 地图功能测试指南

## 高德地图API配置

1. **获取API Key**
   - 访问 [高德开放平台](https://lbs.amap.com/)
   - 注册账号并创建应用
   - 获取Web端JavaScript API Key

2. **配置环境变量**
   ```bash
   # 编辑 .env.development 文件
   VITE_AMAP_KEY=your-amap-key-here
   ```

## 常见问题排查

### 地图不显示
1. 检查API Key是否正确配置
2. 检查网络连接
3. 查看浏览器控制台错误
4. 确认高德地图API加载成功

### 绘制功能异常
1. 检查浏览器是否支持
2. 查看JavaScript控制台错误
3. 确认地图实例初始化成功

### 数据保存异常
1. 检查后端接口返回
2. 查看网络请求状态
3. 确认数据格式正确

## 性能优化建议

1. **地图加载优化**
   - 延迟加载地图组件
   - 缓存地图实例
   - 合理设置地图中心点和缩放级别

2. **绘制性能优化**
   - 限制航点数量（建议不超过50个）
   - 使用防抖处理频繁操作
   - 优化标记点渲染

3. **用户体验优化**
   - 添加加载状态提示
   - 提供操作反馈
   - 支持键盘快捷键

## 部署注意事项

1. **生产环境配置**
   ```bash
   # .env.production
   VITE_AMAP_KEY=your-production-key
   ```

2. **安全配置**
   - 限制API Key使用范围
   - 配置域名白名单
   - 定期更换Key

3. **监控配置**
   - 地图加载成功率监控
   - API调用次数监控
   - 错误日志收集
